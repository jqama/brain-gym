Brain Gym

Welcome to Brain Gym - my personal space for sharpening problem-solving skills and preparing for technical interviews.

This repository contains a structured collection of:
    Coding problems (e.g., LeetCode-style challenges)
    SQL queries and database problems
    Puzzles and brain teasers
    Behavioral interview questions
    Technical questions and concepts

Purpose
    The goal of this repo is simple, stay consistent, think better, and document learning.

Instead of solving problems once and forgetting them, I use this space to:
    Track progress
    Revisit patterns
    Build intuition over time
